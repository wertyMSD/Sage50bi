    Con cuenta Gmail es necesario configurar el API creando un proyecto para alimentar la tablas necesarias esto le permitirá compartir los informes o realizar cuadros de mandos. Estas configuración permiten autenticarnos con Google y acceder a sus servicios de forma segura y eficiente.

    Configuracion

    Para empezar, necesitamos crear un proyecto en la consola de Google Cloud Platform (GCP) y habilitar las APIs de Drive y Sheet. Para ello, seguimos los siguientes pasos:

    • Accedemos a la consola de GCP con nuestra cuenta de Google y creamos un nuevo proyecto o seleccionamos uno existente.

    APIs y servicios habilitados – APIs y servicios – Consola de Google Cloud ![[creacion proyecto api google.png]]

    • En el menú lateral izquierdo, hacemos clic en APIs y servicios y luego en Biblioteca

      • Buscamos las APIs de Drive y Sheet y las habilitamos una por una. Google Sheets API Consola de Google Cloud Google Drive API Consola de Google Cloud

    • En el menú lateral izquierdo, hacemos clic en Credenciales y luego en Crear credenciales.

      • Elegimos la opción ID de cliente de OAuth si queremos usar Oauth 2.0 o
      • Clave de cuenta de servicio si queremos usar .auth_service_google.

    • Seguimos las instrucciones para generar las credenciales y descargar el archivo JSON correspondiente.

    Colocamos en la Subcarpeta SAGE50BI el fichero que generado por API Google
